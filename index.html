What we do with your calendar
What we read
Busy / free blocks only
We see whether a time slot is marked as busy or free. Nothing else.
Date and time of blocks
We need to know when you're unavailable to find windows where the whole group is free.
What we never read
Event titles
"Doctor appointment", "Job interview", "Therapy", we never see any of this.
Event descriptions or notes
Any text you've added to a calendar event is not accessed.
Who the event is with
Guest lists, attendees, organisers, none of this is read or stored.
What we do with the data
Find overlapping free windows
We compare busy/free blocks across your group to find dates when everyone is available. That's the only computation we run.
We never sell calendar data
Your availability is never shared with advertisers, data brokers, or any third party.
We never store raw calendar data
We store only the computed free windows, not the underlying events. Disconnect anytime.
holid.ai uses standard OAuth, the same authentication used by Calendly, Notion, and Zoom. You grant read-only access and can revoke it anytime from your Google, Apple, or Microsoft account settings.
